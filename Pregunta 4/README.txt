### Pregunta 4: Integración de API en JavaScript

## Descripción
Este proyecto muestra cómo integrar la API de Random User en un archivo HTML con JavaScript para mostrar una tabla de usuarios.

## Estructura del Proyecto
- `index.html`: Archivo HTML principal que contiene el código para integrar la API de Random User.

## Requisitos
- Navegador web moderno

## Instrucciones

### Configuración y Uso
1. Abre el archivo `index.html` en tu navegador.
2. Haz clic en el botón "Actualizar" para obtener datos de usuarios de la API de Random User y mostrarlos en la tabla.
